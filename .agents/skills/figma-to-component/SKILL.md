# Figma 设计稿转组件技能

## 核心原则：素材提取优先，禁止 AI 生成素材

本技能分为两个严格分离的阶段：

- 阶段 A：素材提取。纯提取任务，禁止任何生成、重绘、补全或优化行为。
- 阶段 B：组件编码。只使用阶段 A 提取的真实素材组装 Vue 3 组件。

阶段 A 必须在阶段 B 之前完成。未完成素材提取，禁止进入编码阶段。

## 阶段 A：像素级素材提取（强制执行）

### 提取规则

- 这是素材提取任务，不是生成任务。
- 必须直接使用 Figma 设计稿中的真实元素。
- 禁止行为，违反任何一条即视为失败：
  - 重新设计
  - 重新生成
  - 风格参考后重绘
  - AI 补全
  - 自动优化
  - 修改结构
  - 修改颜色
  - 修改发光效果
  - 修改渐变效果
  - 修改透明度
  - 修改阴影
  - 修改边框
- 必须保留：
  - 原始背景色
  - 原始渐变
  - 原始玻璃质感
  - 原始发光
  - 原始边框
  - 原始装饰线条
  - 原始透明层级效果
  - 原始视觉风格
- 必须删除：
  - 所有文字
  - 所有数字
  - 所有图表
  - 所有数据内容
  - 所有图标
  - 所有业务信息

### 裁切要求

- 严格按照元素实际边界进行裁切。
- 不允许额外留白。
- 不允许增加画布。
- 不允许减少有效区域。
- 不允许自动扩展内容。
- 不允许把多个元素合并裁切。

### 尺寸要求

- 输出尺寸必须与原图中该元素实际占用尺寸完全一致。
- 保持原始宽高比例。
- 不允许缩放。
- 不允许拉伸。
- 不允许重新适配尺寸。
- 文件命名应稳定、语义清晰，并与组件用途对应。

### 输出要求

- 每个元素独立输出。
- 每个底部按钮单独输出。
- 一个元素对应一个独立文件。
- 不允许素材拼版。
- 不允许多个素材放在同一张图中。
- 输出到 `figma/exports/` 下对应分类目录。

### 质量要求

- 优先执行像素级提取，通过 Figma MCP 或 Figma 原始导出能力完成。
- 目标是获得设计稿中的真实设计资产，而不是获得新的设计稿。
- 如果无法确认元素边界、图层结构或导出尺寸，必须停止并说明缺口，不得猜测生成。

### 当前提取目标

1. 通用模块背景框（仅提取一个完整模块框）

### 提取输出路径

- 通用模块背景框：`figma/exports/module-frames/module-frame-default.png`

## 阶段 B：组件编码（素材提取完成后执行）

### 工作流程

1. 通过 Figma MCP 调用 `get_design_context` 获取设计稿结构信息。
2. 调用 `get_screenshot` 获取设计截图，保存到 `figma/screenshots/`。
3. 自动提取颜色、字体、间距等设计变量，更新 `src/styles/variables.scss`。
4. 使用阶段 A 已提取的真实素材，即 `figma/exports/` 下的文件，作为组件背景或装饰资产。
5. 根据设计结构生成对应的 Vue 3 组件代码。
6. 对照 Figma 截图检查尺寸、间距、层级和素材引用是否一致。

### 编码规则

- 背景框必须使用 `<img>` 或 `background-image` 引用阶段 A 提取的素材。
- 禁止用 CSS 代码重新绘制背景框的渐变、发光、边框效果。
- 禁止用 CSS 代码模拟玻璃质感或装饰线条。
- 所有视觉装饰效果必须来自提取的真实素材。
- 使用 `<script setup>` + SCSS。
- 颜色、字号使用 `variables.scss` 中的变量。
- 响应式适配基于 1920 x 1080 设计稿，使用 `vw` / `vh` 单位。
- 复制 Figma 链接时，优先复制单个组件或画板链接。
- 组件内可以承载业务数据、标题、数值和图表，但背景装饰必须使用真实切图资产。
- 缺失素材时不得用 CSS 或 AI 生成替代，应保留明确占位并告知用户需要补充导出。

## 防幻觉规则

- 不得根据“看起来像”推断不存在的图层、纹理、图标或装饰。
- 不得把参考截图当作可重新设计的风格板。
- 不得自行更换颜色、光效、边框、圆角、渐变方向或透明度。
- 不得用 CSS、SVG、Canvas 或 AI 图像生成来仿制 Figma 已有素材。
- 不得将文字、数字、图表、图标等业务内容烘焙进背景切图。
- 不确定的地方必须显式标注为待确认，而不是直接实现成确定样式。

## 输出文件

- `src/components/Panels/` 下的对应面板组件
- `src/styles/variables.scss`，用于维护设计变量
- `figma/exports/`，用于保存像素级提取的切图
- `figma/screenshots/`，用于保存自动截图
